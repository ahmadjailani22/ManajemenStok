🛒 toko-safitri-system

Website Sistem Manajemen Barang & Laporan Harian berbasis web untuk Toko Safitri, dibuat menggunakan PHP dan MySQL untuk membantu pengelolaan inventori, transaksi, dan laporan secara efisien.

🏪 SISTEM TOKO SAFITRI

📌 Deskripsi

Sistem ini merupakan aplikasi berbasis web yang dirancang untuk membantu Toko Safitri dalam mengelola:

Stok barang
Transaksi penjualan
Data supplier
Laporan harian & bulanan

Sistem ini menggantikan pencatatan manual agar lebih cepat, akurat, dan terintegrasi.

🚀 Fitur
🔐 Autentikasi
Login & Logout
Manajemen role (Admin, Kasir, Pemilik)
📦 Manajemen Produk
Tambah, edit, hapus produk
Kelola kategori
Upload foto produk
📊 Inventori
Update stok masuk & keluar
Notifikasi stok minimum
Histori perubahan stok
🛒 Point of Sale (POS)
Transaksi penjualan
Riwayat transaksi
🚚 Supplier
Data supplier
Riwayat pembelian
📈 Laporan
Laporan penjualan harian
Laporan bulanan
Produk terlaris
Export PDF / Excel
⚙️ Teknologi
Frontend: HTML, CSS, JavaScript, Bootstrap 5
Backend: PHP
Database: MySQL
Server: Apache (XAMPP)
Version Control: GitHub
